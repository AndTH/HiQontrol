HiQontrol
=========

Soundraft Si Compact 16 Harman HiQnet Controller

Status
======

Work In Progress, [Proof of Concept](https://github.com/EMATech/HiQontrol/releases/tag/v0.0.1) available for Android

Dependencies
============

- Python 2 or 3
- [Kivy](kivy.org)
- netifaces
- twisted
